🚀 About Open Omniscience

Open Omniscience is a powerful, ethically-designed platform that aggregates raw news information from diverse worldwide sources into a unified, highly searchable, and auditable database. Built as a fork of HTTrack, it leverages robust crawling capabilities while adding advanced features for ethical scraping, duplicate detection, and comprehensive data management.

Our mission is to empower investigative journalism by enabling users to cross-reference disparate pieces of information, identify complex patterns and disinformation schemes, and preserve data integrity for accountability.

✨ Core Features

Global Scraping

Ingest articles from 1900+ predefined news sources using both RSS feeds and HTML scraping with full ethical compliance.

Ethical Scraping

Respects robots.txt directives, implements rate limiting, and uses proper User-Agent identification for responsible scraping.

Duplicate Detection

Advanced URL canonicalization and SHA-256 content hashing ensures no duplicate articles are stored in your database.

Flexible Storage

Supports both SQLite (portable, default) and PostgreSQL (scalable, production-ready) database backends.

Advanced Search

Full-text search with Boolean operators (AND, OR, NOT), filters by date, source, language, and tags with pagination.

Source Management

Comprehensive source management with groups, tags, metadata, and batch operations for easy organization.

RSS Discovery

DuckDuckGo-powered RSS feed discovery automatically finds missing feeds for your news sources.

Statistics Dashboard

Visual statistics dashboard showing source counts, group information, and scraping activity metrics.
